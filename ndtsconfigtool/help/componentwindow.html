The component window can be opened either by selecting one of the components from the dock window or by choosing the edit action, e.g. from the file menu.

The component windows contains two frames:

  • the left one with Tree View of the selected component,
  • the right one with parameters of the currently selected Component Item in the tree view.

<>p

To make desiging easier there are provided buttons: Apply and Reset related to the selected component item and Save and Close for the current component.
