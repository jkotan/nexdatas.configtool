File|New Create a new component.

File|New DataSource Create a new data source.

File|Open Open an existing component.

File|Open DataSource Open an existing data source.

File|Edit Edit the current component.

File|Edit DataSource Edit the current data source.

File|Save Save the current component.

File|Save DataSource Save the current data source.

File|Save As Save the current component under a new name, and make the newly named component the current component.

File|Save DataSource As Save the current data source under a new name, and make the newly named datasource the current component.

File|Save All Save all components.

File|Save All DataSource Save all data sources.

File|Close Remove the current component from the component list.

File|Close DataSource Remove the current data source from the data source list.

File|Reload List Reload the component list from the current component file directory.

File|Reload DataSource List Reload the data source list from the current datasource file directory.

File|Change Directory... Change the current component file directory and Reload the component list from the directory.

File|Change DataSource Directory... Change the current data source file directory and Reload the data source list from the directory.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.
