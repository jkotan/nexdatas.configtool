The NDTS Component Designer program allows creates XML components as well as datasources stored in Configuration Tango Server and used by Nexus Data Writer (NDTS).

The File menu is used to create new or load existing components and datasources, and to save the currently edited files in files.

The Edit menu offers the cut copy paste and apply options of edited component items.

The Component Items menu contains adding new or loading existing component items as well as component merging and clearing.

The options in the Server menu allow to connect to the configuration server and fetch or store the created configuration files on it.

The View menu offers switches to show and hide the collection dock window with lists and to change display of the component tree columns.

In the Window menu one can find standard window operations for applications with the multiple document interface.

The Help menu offers informations about the Component Designer version as well as the current detail help.

Collection Dock Window contains lists of currently open components and datasources. Selecting component or datasource from the lists one can open either Component Window or DataSource Window.

All the most commonly used menu options are also available on Toolbar.
