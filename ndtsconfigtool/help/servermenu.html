Server|Connect Open the dialog with access parameters of Configuration Tango Server. After specifying those parameter one can connect into the server.

Server|Fetch Components Fetch the components from the Configuration Server.

Server|Save Component Store the current component in the Configuration Server.

Server|Delete Component Delete the current component from the Configuration Server.

Server|Fetch DataSources Fetch the datasources from the Configuration Server.

Server|Save DataSource Store the current datasource in the Configuration Server.

Server|Delete DataSource Delete the current datasource from the Configuration Server.

Server|Get Mandatory Components Display names of the component stored on the Configuration Server selected as mandatory.

Server|Set Component Mandatory Set the current component as mandatory on the Configuration Server.

Server|Unset Component Mandatory Set the current component as non-mandatory on the Configuration Server.

Server|Close Close the connection with the Configuration Server.
