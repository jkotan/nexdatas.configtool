Window|Next Select the next window.

Window|Next Select the previous window.

Window|Cascade Rearrange the open windows in the cascade structure.

Window|Tile Rearrange the open windows in the tile structure.

Window|Restore All Reopen all minimised windows.

Window|Iconize All Minimise all the windows.

Window|Close Close the selected window.

On the last positions of the window menu one can also select the currently open or iconized windows.
