Edit|Undo Undo the last executed command.

Edit|Redo Undo the last un-executed command.

Edit|Cut Move the selected item into the clipboard.

Edit|Copy Copy the selected item into the clipboard.

Edit|Paste Paste the item from the clipboard into the selected item.

Edit|Cut Component Item Move the selected component item into the clipboard.

Edit|Copy Component Item Copy the selected component item into the clipboard.

Edit|Paste Component Item Paste the item from the clipboard into the selected component item.

Edit|Cut DataSource Move the selected data source into the clipboard.

Edit|Copy DataSource Copy the selected data source into the clipboard.

Edit|Paste DataSource Paste the item from the clipboard into the selected data source.

Edit|Edit Component Edit the selected component.

Edit|Take DataSource Item Takes the selected data source in the component tree to a separate datasource window.

Edit|Take DataSources Takes all data sources from the the current component to separate datasource windows.

Edit|Edit DataSource Edit the selected data source.

Edit|Apply DataSource Apply the selected data source.
