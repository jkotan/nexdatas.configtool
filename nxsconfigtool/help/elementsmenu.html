Elements|New Component Create a new component.

Elements|Edit Component Edit the current component.

Elements|Merge Component Merge the current component and check correctness of its structure.

Elements|Close Component Remove the current component from the component list.

Elements|New DataSource Create a new data source.

Elements|Edit DataSource Edit the current data source.

Elements|Apply DataSource Apply the selected data source.

Elements|Close DataSource Remove the current data source from the data source list.

Elements|Quit Terminate the program. You will be given the chance to save any unsaved changes.
