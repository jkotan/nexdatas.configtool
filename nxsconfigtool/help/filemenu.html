File|Load Component Open an existing component.

File|Load DataSource Open an existing data source.

File|Save Component Save the current component.

File|Save DataSource Save the current data source.

File|Save Component As Save the current component under a new name, and make the newly named component the current component.

File|Save DataSource As Save the current data source under a new name, and make the newly named datasource the current component.

File|Save All Components Save all components.

File|Save All DataSources Save all data sources.

File|Reload Component List Reload the component list from the current component file directory.

File|Reload DataSource List Reload the data source list from the current datasource file directory.

File|Change Directory... Change the current component file directory and Reload the component list from the directory.

File|Change DataSource Directory... Change the current data source file directory and Reload the data source list from the directory.
